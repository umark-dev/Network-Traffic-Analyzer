# Network Traffic Analyzer

## Description
This project simulates packet-level traffic analysis by inspecting network logs and identifying suspicious protocols, ports, and abnormal traffic patterns.

## Features
- Detects insecure protocols (FTP, Telnet)
- Monitors UDP traffic (SNMP)
- Identifies high traffic volume (possible scanning/flooding)
- Generates alert reports

## Technologies
- Python
- Network Security
- Traffic Analysis
- SOC Monitoring

## How to Run
python analyzer.py

## Output
Alerts saved in alerts.txt
